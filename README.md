# "e-plantShopping fork"

Este repo es un fork de: ['e-plantShopping'](https://github.com/ibm-developer-skills-network/e-plantShopping.git).

Este "source code" diseña la página de compras para un negocio de venta de plantas.

La aplicación web incluye:
- Landing Page
- Catálogo de Productos
- Carrito de compras

Se emplean las librerías `React` (UI) y `Redux Toolkit` (gestión de datos y eventos).
